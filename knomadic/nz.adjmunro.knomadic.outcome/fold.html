fold

inline suspend fun <In : Any, Out : Any, ErrorIn : Any, ErrorOut : Any> Outcome<In, ErrorIn>.fold(crossinline success: suspend (In) -> Out, crossinline failure: suspend (ErrorIn) -> ErrorOut): Outcome<Out, ErrorOut>

Returns a new Outcome, after applying success or failure based on the current Outcome state.

  • Transforms Outcome<In, ErrorIn> into Outcome<Out, ErrorOut>.

  • This function does not provide a RaiseScope, and makes no guarantees about catching, handling, or rethrowing errors! Use outcomeOf within the transformation lambdas for that.

  • Unlike Outcome.flatFold, fold's transform lambdas return the monad's internal value directly instead of the Outcome wrapper.

Receiver

The Outcome<In, ErrorIn> to transform.

Return

A new Outcome<Out, ErrorOut> with the transformed value.
